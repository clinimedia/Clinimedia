How to use these

Each template lives in its own folder under templates/ and is fully self-contained (HTML + CSS + JS). To customize for a client: open the folder, edit the text in index.html, and change the colors at the top of styles.css (look for the :root variables). No build tools or installation required — just open the file in a browser. See README.md for full details.
